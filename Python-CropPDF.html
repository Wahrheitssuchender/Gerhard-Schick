PDF Text Extraction and Cropping

SITUATION: An e-commerce platform user sells items to consumers and manually packs and sends items to the buyers. PDf files contaiing the airway bills (AWBs) and corresponding packing lists (PLs) are printed for each purchase.
COMPLICATION: AWBs and PLs are printed automatically on the same printer with adhesive labels despite only the AWB needs to be applied to the delivery package, which results in unneccessary waste of labels.
SOLUTION: The original PDF is split into AWBs and PLs. The information of the PLs is extracted and a separate set of PDFs is generated that are size optimized to be printed on a separate label printer.
