MS Access/VBA/MySQL

Order/Inventory/Customer Management solution.

Set up for previous employer to handle back-end processes.

SITUATION: The company handles B2B wholesale trades of a wide range of FMCG products.
COMPLICATION: Order processing was done manually without any database in place to manage customer, product and shipment data. This caused unnecessary delays and errors along the process until the final order delivery. Required data was spread over different files and locations adding to the problem.
SOLUTION: MS Access database that organizes all relevant data and allows to create invoices, purchase and delivery orders. Product, customer, shipping, inventory data is stored on a MySQL backend database. Integration with Outlook allows automated sending of documents to customers and suppliers. Currency exchange rates are retrieved via an API to allow for accurate stock value and price calulations. Number of design and code elements were taken over and adapted from an existing database solution.
