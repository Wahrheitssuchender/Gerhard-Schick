Data Scraping

SITUATION: A premium wine retailer manages business data in a database which includes customer, sales and product data. Data analysis and visualization is done with state-of-the-art BI visualization tools.
COMPLICATION: The product data is fairly basic and does not allow for more complex analysis such as product and customer clustering.
SOLUTION: Creation of an automated web scraping flow utilizing Make.com which initiates API calls to an AI LLM model to gather additional data to allow further product and customer characterization. Results are stored in a cloud spreadsheet that can be queried by the ERP for data merging.
