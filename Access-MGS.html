MS Access/VBA

Application to calculate profit sharing.

Designed and implemented for an industry client.

SITUATION: The client provides customized merchandise items and sources them made to order from different overseas vendors. Sales representatives and agents are compensated on a commission base.
COMPLICATION: To calculate commissions for each several transactions during the order cycle have to be captured, which involves manual processing, takes extensive amoun of time and is prone to errors.
SOLUTION: MS Access application that tracks all orders and allows to import order transactions from an Excel sheet to calculate the appropriate profit share for each stakeholder. FX rates can be set for each month to account for exact currency conversion.
